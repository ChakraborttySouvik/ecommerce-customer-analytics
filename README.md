# E-commerce Customer Analytics

## 📌 Project Overview

This project focuses on analyzing E-commerce retail transactions to extract business insights related to customer behavior, product performance, sales trends, and revenue distribution.

The analysis combines Python-based exploratory data analysis, customer segmentation, forecasting techniques, and an interactive Power BI dashboard.

---

## 🎯 Objectives

* Perform customer segmentation using RFM Analysis.
* Analyze customer retention using Cohort Analysis.
* Classify products using ABC Inventory Analysis.
* Identify top-performing products.
* Analyze transaction and sales patterns.
* Forecast future sales trends.
* Build an interactive Power BI dashboard for business reporting.

---

## 🛠️ Technologies Used
